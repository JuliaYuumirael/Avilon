Заметки разработчика (Хельмдора SPA v5 - Система Заметок):

Палитра: Динамически генерируется или выбирается из предопределенных (Архидемоны + Случайная) для каждого класса.

Структура: SPA с графом знаний на основе классов. Chart.js заменен на редактор заметок (contenteditable div) для каждого узла. Добавлены кнопки форматирования (Жирный, Курсив, Подчеркнутый). Содержимое заметки (HTML) сохраняется в localStorage. ID Узлов генерируются автоматически. Функции удаления узлов и классов с подтверждением.

Визуализация: Редактор заметок `contenteditable div`. Кнопки форматирования. Модель данных: узлы теперь хранят `noteHTML` для форматированного текста. Обоснование: Реализация запроса пользователя на систему заметок, замена менее релевантного графика.

Подтверждение: SVG/Mermaid не используются. Chart.js удален.
